# Production Working Hours Analysis Dashboard
Interactive Production Working Hours Analysis Dashboard to analyse workforce productivity and labour costs.
## Features

- KPI Cards
- Interactive Filters
- Division Analysis
- Employee Analysis
- Overtime Analysis
- Budget vs Actual
- Dark Mode
- Responsive Design

## Skills Demonstrated

- Dashboard Design
- Business Intelligence
- Financial Analytics
- Data Visualization
